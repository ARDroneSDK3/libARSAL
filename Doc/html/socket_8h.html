int sal_connect ( int  sockfd,
const struct sockaddr *  addr,
int  addrlen 
)

Initiate a connection on a socket, If the socket sock is of type SOCK_TYPE_DGRAM then addr is the address to which datagrams are sent by default, and the only address from which datagrams are received. If the socket is of type SAL_SOCK_TYPE_STREAM, this call attempts to make a connection to the socket that is bound to the address specified by addr.

Parameters:
sockfdThe socket to connect
addrThe address to connect.
addrlenThe size of the addr
Return values:
Onsuccess, 0 is returned. Otherwise, -1 is returned and errno is set appropriately. (See errno.h)
